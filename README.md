
# Telegram Bot GetCID - Versión Lite (Render Free Compatible)

Este bot de Telegram detecta ID de instalación desde imágenes usando Tesseract OCR (compatible con 512 MB de RAM).

## ✅ Cómo desplegar en Render

1. Sube este código a GitHub.
2. En Render.com, crea un nuevo Web Service desde GitHub.
3. Usa:
   - Build command: `pip install -r requirements.txt`
   - Start command: `python main.py`
4. Agrega variable de entorno:
   - `BOT_TOKEN` = tu token del bot de Telegram

¡Y listo!
